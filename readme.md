## Getting Started

This is an example of how you may give instructions on setting up your project locally.
To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

- npm
  ```sh
  npm install npm@latest -g
  ```
- For Clean Install (Sorry For Missing)
  ```sh
  npm install mysql2
  npm install express
  npm install discordjs
  npm install axios
  npm
  ```

### Installation

1. Get a token Key at [https://discord.com/developers/applications](https://discord.com/developers/applications)
2. Clone the repo
   ```sh
   git clone https://github.com/AkmalYonan/HIPEACE.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter Key API in `config.js`
   ```js
    "token": "TOKEN-BOT-HERE", // BOT DISCORD APPLICATION KEY TOKEN
    "clientId": "927558297211514890", //TESTING (ID User Discord)
    "guildId": "862907253631483915", //TESTING (Guild = Server Discord)
    "tripayMode": "sandbox", //MODE TRIPAY
    "tripayApiKey": "TOKEN-TRIPAY", // TOKEN API KEY TRIPAY
    "tripayPrivateKey": "TOKEN-PRIVATE-TRIPAY", // TOKEN PRIVATE KEY TRIPAY
    "tripayMerchantCode": "SANDBOX-MERCHANT" // MERCHANT CODE TRIPAY
   ```
5. Change git branch to avoid accidental pushes to base project

   git remote set-url origin AkmalYonan/repo_name
   git remote -v # confirm the changes
   ``` -->
